* Usage
Working directory is assumed to be the project root.
** Get datasets
*** Download datasets from external sources
#+begin_src sh
poetry run python -m learn2rag.evaluation.downloader
#+end_src

*** Download datasets from our server
https://files.dice-research.org/projects/Learn2RAG/datasets/

Save to ~./datasets~.
** Get the prepared document database
https://files.dice-research.org/projects/Learn2RAG/evaluation/qdrant_data.zip

Unpack to ~./learn2rag/evaluation/qdrant_data~.
** Start the document database
#+begin_src sh
docker compose -f learn2rag/evaluation/docker-compose.yml up -d qdrant
#+end_src
** Import datasets
#+begin_src sh
poetry run python -m learn2rag.evaluation.importer
#+end_src
** Ingest
GPU recommended.

#+begin_src sh
poetry run python -m learn2rag.evaluation.ingester
#+end_src
** Example queries from datasets
#+begin_src sh
poetry run python -m learn2rag.evaluation.example
#+end_src
** Stop the document database
#+begin_src sh
docker compose -f learn2rag/evaluation/docker-compose.yml down qdrant
#+end_src
