* Learn2RAG basic implementation
** Requirements
*** Python
*** Ollama
*** Docker
*** direnv
** Install
*** Packages
#+BEGIN_SRC sh
  apt install build-essential python3-venv docker.io docker-compose direnv
  direnv hook bash >>~/.bashrc
#+END_SRC
*** Ollama
#+BEGIN_SRC sh
curl -L https://ollama.com/download/ollama-linux-amd64.tgz -o ollama-linux-amd64.tgz
tar -xzvf ollama-linux-amd64.tgz
bin/ollama serve
#+END_SRC
#+BEGIN_SRC sh
curl http://localhost:11434/api/pull -d '{"model": "llama3.3:70b"}'
#+END_SRC sh
*** Learn2RAG
#+BEGIN_SRC sh
  make
#+END_SRC
Configure options in ~./.local.envrc~:
#+BEGIN_SRC sh
export OLLAMA_URL="..."
export OLLAMA_AUTH="Bearer ..."
#+END_SRC
You might need to run ~direnv allow~ in the same directory.
** Run
*** Run an example with in-memory index store
#+BEGIN_SRC sh
  ./main.py <<< '@scripts/example-inmemory-run'
#+END_SRC
*** Run an example with persistent vector store
#+BEGIN_SRC sh
  # Start the database
  docker-compose up -d redis
  # Index documents once
  ./main.py <<< '@scripts/index-aiact'
  # Run queries using the index
  ./main.py <<< '@scripts/query-aiact'
  ./main.py <<< '@scripts/query-aifaq'
#+END_SRC
*** Start and interactive session
#+BEGIN_SRC sh
  ./main.py
#+END_SRC
#+BEGIN_SRC
  use vector_store RedisVectorStore aiact
  build
  query '"Was ist KI-Gesetz?"'
#+END_SRC
*** Start web interface
**** Initial setup
#+BEGIN_SRC sh
  # Index documents
  ./main.py <<< '@scripts/index-aiact'
#+END_SRC
Configure options in ~./.env~:
#+BEGIN_SRC sh
  OLLAMA_URL="..."
  OLLAMA_AUTH="Bearer ..."
  REDIS_INDEX_NAME=aiact
#+END_SRC
#+BEGIN_SRC sh
  # Start all components
  docker-compose up -d
#+END_SRC

Open [[http://localhost:3000/][Open WebUI]], set up a local admin account.

In user menu, Settings, Admin Settings, Connections, OpenAI API, "+" set URL to ~http://open-webui-pipelines:9099~ (without the trailing slash) and API key to ~0p3n-w3bu!~.

In user menu, Settings, Admin Settings, Pipelines set Api Uri to ~http://langgraph:9000/stream~.
